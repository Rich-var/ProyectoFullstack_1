# ProyectoFullstack_1

## Descripción
Sistema de microservicios desarrollado bajo una arquitectura modular para la gestión integral de operaciones empresariales.

## Integrantes
Sebastián Lillo y Ricardo Vargas.

## Estructura de Servicios
El proyecto se compone de los siguientes módulos independientes:

- **Auth**: Gestión de seguridad y autenticación.
- **categorias**: Operaciones relacionadas con la clasificación de productos.
- **empleados**: Gestión de datos y registros del personal.
- **inventario**: Administración de existencias y stock.
- **notificaciones**: Sistema de envío y gestión de alertas.
- **pedidos**: Procesamiento y seguimiento de órdenes.
- **productos**: Gestión del catálogo de productos.
- **proveedores**: Administración de los proveedores del sistema.
- **sucursales**: Gestión de los puntos físicos o lógicos de la empresa.
- **usuarios**: Administración de perfiles y usuarios del sistema.

## Tecnologías
- **Lenguaje**: Java
- **Arquitectura**: Microservicios
